Формулы для y и x могут содержать следующие выражения:

  • Математические знаки (+-/*)
  • Переменные. Для y= это x. Для x= это y
  • Математические формулы

Доступные математические знаки:

  • + сложение
  • - вычитание
  • * умножение
  • / деление
  • ** или ^ возведение в степень
  • // целочисленное деление
  • % остаток от деления

Доступные математические формулы:

Тригонометрические функции:

  • Классические: sin(v), cos(v), tan(v)
  • Обратные классические: asin(v), acos(v), atan(v)
  • Гиперболические: sinh(v), cosh(v), tanh(v)
  • Обратные гиперболические: asinh(v), acosh(v), atanh(v)

Экспоненциальные и логарифмические функции

  • exp(v) экспонента e**v
  • log(v) натуральный логарифм, log(x) = ln(x)
  • log10(v) десятичный логарифм
  • log2(v) логарифм по основанию 2
  • logb(v, base) логарифм по основанию base

Степенные и корневые функции

  • sqrt(v) квадратный корень
  • power(v, st) возведение в степень v**st

Округления

  • floor(v) округляет число до ближайшего меньшего целого
  • ceil(v) округляет число до ближайшего большего целого
  • round(v) округляет число до ближайшего целого

Специальные функции

  • sign(v) Возвращает знак числа: 1 если v > 0, 0 если v = 0, -1 если v < 0
  • gamma(v) гамма-функция
  • erf(v) вычисляет функцию ошибок
  • hypot(v1, v2) длина гипотенузы: sqrt(v1**2 + v2**2)
  • abs(v) абсолютное значение (модуль)
